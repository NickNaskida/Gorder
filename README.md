# GoTrim

A simple web API service for shortening URLs written in Go and Gin.

### Prerequisites

- Go 1.20 or later

### Installation
1. Clone the repository
    ```sh
    git clone https://github.com/NickNaskida/Gorder.git
    ```
2. Build the project
    ```sh
    go build cmd/Gorder/main.go
    ```
3. Run the executable
